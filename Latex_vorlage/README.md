# Willkommen

Ich stelle euch verschiedene LaTeX Vorlagen bereit. Diese Vorlagen entsprechem dem gängigen deutschen, im Buchdruck üblichen, Layout und sollten für Abgaben wie Hausarbeiten oder Präsentationen geeignet sein.
Der Code ist rudimentär kommentiert und kann über das enthaltende Skript gebaut werden. Sowohl in den Vorlagen, als auch in der beispiele.tex, finden sich kleine Vorlagen für Tabellen, Listings, Abbildungen, Zitate und Funktionen.<br> 
Die Vorlagen können über den oben stehenden Link als tar heruntergeladen werden. Alternativ steht der Weg über wget bzw. curl bereit.  
Zusätzlich existiert ein kurzes How-to welches die ersten Schritte in LaTeX erleichtert.


<center>
Download (cli)
</center>


```bash
wget https://gitlab.informatik.hs-bremerhaven.de/flignitz/latex_vorlagen/-/archive/1.0/latex_vorlagen-1.2.tar
```
<center>
tar entpacken
</center>


```bash
tar -xf latex_vorlagen-1.2.tar
```
